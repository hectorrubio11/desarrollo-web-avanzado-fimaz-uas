Objetivo: Aplicar los fundamentos de la Programación Orientada a
Objetos en PHP, implementando una clase con atributos privados,
constructor y métodos de acceso, siguiendo buenas prácticas de
encapsulamiento.

Para esta práctica, creamos una clase Usuario con las funciones setters y getters de sus dos atributos privados: nombre y correo.
Lo único que se necesita para que funcione es instanciar la clase en el index, darle valores a estos atributos con las funciones setters e imprimirlas con las funciones getters.
